# AngularでDjango REST Framework(API)からデータを取得する勉強

## Angular(ng-playlsit)

```shell
$ yarn start
# フロント・・・localhost:4200に接続
```

## Django(dj_playlist)

```shell
$ python manage.py runserver
# API・・・localhost:8000/playlist/index/に接続
```
